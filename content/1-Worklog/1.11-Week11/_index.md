### Week 11 Achievements:

* Understand how to use VPC Endpoints to establish secure connections between a VPC and AWS services without routing traffic through the public Internet.
* Understand the role of AWS WAF in protecting web applications, and practice configuring Web ACLs and basic rules to control incoming traffic and prevent common web attacks.
* Create and successfully deploy an AWS Lambda function, test it using event triggers, and monitor its execution through Amazon CloudWatch Logs.
* Successfully deploy the project.
