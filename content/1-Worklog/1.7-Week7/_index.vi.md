### Kết quả đạt được tuần 7:

* Cài đặt thành công AWS CLI trên hệ điều hành Windows và cấu hình thông tin xác thực (Access Key ID, Secret Access Key, Region và Output Format) bằng lệnh aws configure, giúp kết nối thành công với tài khoản AWS.
* Thực hành sử dụng các lệnh AWS CLI để quản lý tài nguyên AWS, bao gồm:
  * Liệt kê danh sách S3 Bucket.
  * Tạo Bucket mới.
  * Upload tệp lên S3.
  * Đồng bộ thư mục lên S3.
  * Xóa tệp và Bucket khi không còn sử dụng.
* Thực hiện thành công các thao tác quản lý tài nguyên AWS thông qua giao diện dòng lệnh thay vì AWS Management Console.
* Hiểu được cách AWS CLI hỗ trợ tự động hóa các tác vụ quản lý tài nguyên.
