### Week 9 Objectives:

* Learn about AWS Identity and Access Management (IAM).
* Practice creating IAM Groups, IAM Users, and IAM Roles.

### Week 9 Achievements:

* Successfully create IAM Groups and assign appropriate IAM Policies to grant permissions based on the practice requirements.
* Create IAM Users and add them to IAM Groups so they inherit the configured access permissions.
* Create IAM Roles and assign them to AWS services to grant access to AWS resources without using Access Keys.
* Practice assigning, modifying, and revoking permissions for IAM Users, IAM Groups, and IAM Roles to gain a better understanding of AWS access management.
* Understand the differences between IAM Users, IAM Groups, and IAM Roles, as well as how to apply each of them in real-world access management scenarios.
