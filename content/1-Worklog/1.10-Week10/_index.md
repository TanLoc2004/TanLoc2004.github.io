### Week 10 Objectives:

* Learn to manage systems using AWS Systems Manager.
* Use Amazon CloudWatch to monitor and manage system performance.
* Learn about the application deployment process.

### Week 10 Achievements:

* Successfully manage Amazon EC2 instances using AWS Systems Manager, including connecting to instances, executing remote commands, and performing system administration tasks without using SSH.
* Monitor Amazon EC2 performance metrics in Amazon CloudWatch, such as CPU Utilization, Network In/Out, and Disk Read/Write, and create dashboards to visualize system performance in real time.
* Configure Amazon CloudWatch Alarms to automatically detect when resource usage exceeds predefined thresholds and send notifications for timely incident response.
* Understand the application deployment process on AWS, from environment preparation and service configuration to post-deployment monitoring and operations.
